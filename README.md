# 🎬 Movie Booking App

## 🚀 Tech Stack
- Frontend: Angular + Tailwind CSS
- Backend: Node.js + Express + TypeScript
- Database: MongoDB Atlas
- Auth: JWT Authentication

## ✨ Features
- User Register/Login
- JWT Authentication
- Role-based Access (Admin)
- Movie Listing
- Add Movies (Admin)
- Protected Routes
- Responsive UI

## 📂 Project Structure
frontend/
backend/

## 🏃‍♂️ Run Backend

cd backend
npm install
npm run dev

## ▶️ Run Frontend

cd frontend
npm install
ng serve

## 🔐 Environment Variables

Create .env in backend:
- PORT=5000
- MONGO_URI=your_mongo_uri
- JWT_SECRET=your_secret
